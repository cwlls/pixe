### Key Improvements

1. **Flag Consistency:** All commands now use `--dest` / `-d` for destination directories, matching the `sort` command.
2. **Config File Support:** Removed `MarkFlagRequired` to allow `dest:` in `.pixe.yaml` or `PIXE_DEST` env var to satisfy the requirement.
3. **Copyright Token Syntax:** Replaced `{{.Year}}` with `{year}` syntax, matching path template tokens. Supports `{year}`, `{month}`, `{monthname}`, `{day}`.
4. **Output Formatting:** Sort and status output now prefix destination paths with `...<basename>/` for clarity (e.g., `...Photos/2026/01-Jan/photo.jpg`).
5. **Startup Validation:** Copyright templates are parsed once at startup, avoiding repeated parsing and catching syntax errors early.
